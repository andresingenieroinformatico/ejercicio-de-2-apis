# 🚀 Dashboard Multi-API: Explorador de Datos Vivos

Este proyecto es una aplicación web interactiva desarrollada para la asignatura de **Desarrollo Web II** (VII Semestre) en **UNIPAZ**. El objetivo principal es demostrar la integración y el consumo de múltiples APIs mediante JavaScript moderno (`async/await`).

## 🛠️ Tecnologías Utilizadas
* **HTML5 / CSS3:** Estructura y diseño responsivo.
* **Tailwind CSS:** Framework de estilos para una UI moderna y limpia.
* **JavaScript (ES6+):** Lógica asíncrona mediante el uso de `fetch`.
* **APIs de Terceros:** Consumo de datos reales en formato JSON.

---

## 🌐 APIs Integradas
En este dashboard se han integrado dos APIs distintas para ofrecer una experiencia completa

1.  **Rick and Morty API (Sin clave):** Se utiliza para explorar el multiverso de personajes. Permite realizar búsquedas por nombre y visualizar estados (Vivo, Muerto, Desconocido).
2.  **OpenWeatherMap (Requiere API Key):** Un widget meteorológico que muestra la temperatura y el estado del clima de cualquier ciudad ingresada por el usuario en tiempo real.

---

## ✨ Características Principales
Siguiendo los requisitos del proyecto, la aplicación incluye:
* **Interfaz Agradable:** Diseño de dashboard de dos columnas.
* **Interacción del Usuario:** Buscador de personajes y filtro de ciudades por nombre.
* **Manejo de Estados:** Indicadores de "Cargando..." mientras se obtienen los datos.
* **Manejo de Errores:** Notificaciones visuales si la ciudad no existe o la API falla.
